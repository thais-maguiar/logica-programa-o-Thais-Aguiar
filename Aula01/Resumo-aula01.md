## 1. O que é Algortimo? 
##É uma seqência lógica, finita que serve para resolver problemas ou realizar alguma tarefa específica.
## 2. A estrutura básica 
##Entrada: Onde se recebe os dados(ex: nome dos alunos e notas)
##Processamento: Manipulação dos dados, o que o programa irá fazer com os dados (ex: organizar os nomes e as notas dos alunos )
##Saída: Mostra o resultado do que foi feito. (ex: mostrar o valor das notas dos alunos e seus nomes.)
## 3. Exemplo do dia a dia: Arrumar-se para ir ao trablho 
   1- Levantar da cama
   2- Preparar o café
   3- Tomar banho
   4- Vestir a farda 
   5- Tomar café
   6- Escovar os dentes 
   7- Pegar a bolsa/mochila
   8- Ir para o trabalho

## 4. 5 Palavras-Chave:
    1.**Fluxograma**: Representação visual (gráfica) de um algoritmo usando símbolos e setas.
    2. **Algoritmo**: Uma sequência de passos bem definidos para resolver um problema.
    3.**Pseudocódigo**: Uma forma de escrever algoritmos usando uma linguagem próxima da nossa (ex: "Portugal"), antes de passar para o código real.
    4. **Variável**: Espaço na memória para guardar um dado.
    5**Lógica**: Um pensamento estruturado para resolver o problema.
